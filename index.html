WEBサイトの作り方とは？

学ぶべきプログラミング言語

プログラミング言語 役割 特徴
HTML Webページの「骨組み」を作る言語
  • 見出し・段落・表・リンク・画像などの要素を配置できる
  • 「タグ」を使って文書構造を定義する
  • データやコンテンツを表現するが、デザインや動作は扱わない
CSS Webページの「見た目・デザイン」を指定する言語
  • 文字の大きさ、色、フォント、余白、背景、レイアウトなどを調整できる
  • HTMLに装飾を与えて、デザインを統一する
  • 「セレクタ」で要素を指定し、「プロパティ」でデザインを設定する
  • レスポンシブデザイン（スマホやタブレット対応）も可能
JavaScript Webページに「動き・機能」を追加する言語
  • ボタンを押したら表示が変わる、フォームの入力チェック、アニメーションなどを実現
  • HTMLやCSSを操作して、ページを動的に変化させられる
  • Web APIと連携してデータを取得・表示できる
  • サーバーサイド（Node.js）でも利用可能
